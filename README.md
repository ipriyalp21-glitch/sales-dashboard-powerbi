# 📊 Sales Performance Dashboard (Power BI)

## 📌 Objective
To analyze retail sales data and generate insights for business decision-making.

## 🛠 Tools Used
- Power BI  
- Excel  

## 📊 Dataset
- Superstore dataset (1000+ records)

## 📈 Key Insights
- West region generated highest sales  
- Technology category contributed most to revenue  
- Sales trends increased over time  

## ✅ Outcome
Built an interactive dashboard to track KPIs like total sales, orders, and category performance.
